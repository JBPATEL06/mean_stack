# Common Semantic HTML Tags

### Page Structure

```html
<header>
<nav>
<main>
<section>
<article>
<aside>
<footer>
```

---

### Text Content

```html
<h1> to <h6>
<p>
<blockquote>
<pre>
<hr>
```

---

### Lists

```html
<ul>
<ol>
<li>
<dl>
<dt>
<dd>
```

---

### Links & Navigation

```html
<a>
```

---

### Media

```html
<figure>
<figcaption>
<img>
<audio>
<video>
<track>
```

---

### Tables

```html
<table>
<caption>
<thead>
<tbody>
<tfoot>
<tr>
<th>
<td>
```

---

### Forms

```html
<form>
<label>
<fieldset>
<legend>
<input>
<textarea>
<select>
<option>
<button>
<output>
<datalist>
```

---

### Text Meaning

```html
<strong>
<em>
<mark>
<abbr>
<cite>
<code>
<kbd>
<samp>
<var>
<time>
<address>
<dfn>
<del>
<ins>
<small>
<sub>
<sup>
```

---

### Interactive Elements

```html
<details>
<summary>
<dialog>
```

---

# Non-Semantic Tags

```html
<div>
<span>
```

These provide **no meaning**, only grouping/styling.

---

```html
<header>

<article>
<aside>
<footer>
<figure>
<figcaption>
<form>
<label>
<fieldset> 
<legend>

```

These are the semantic tags interviewers most often expect you to know.
